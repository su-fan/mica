# mica-metrics 组件

## 功能
- druid 指标收集。
- undertow 指标收集。
- sentinel 指标收集。

## 使用
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-metrics</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-metrics:${version}")
```
